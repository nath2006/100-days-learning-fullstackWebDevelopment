# Bacis Unix Command Line
## Make New and Add Folder
> ```
> //syntax
> mkdir [folder_name]
>
>//ex
>mkdir project-Mobile
> ```
## Remove Folder
> ```
> //syntax
> rmdir [folder_name]
>
>//ex
>rmdir project-Mobile
> ```
## Make New and Add Fie
> ```
> //syntax
> touch [file_name]
>
>//ex
>touch index.html
> ```
## Remove Fie
> ```
> //syntax
> rm [file_name]
>
>//ex
>rm index.html
> ```
## List Directory and File
> ```
> //syntax
> ls
>
>//ex
> ls
> ``
## Change directory
> ```
> //syntax
> cd [directory_name/]
>
>//ex
> cd D-18\(SASS-1\)/
> ``
